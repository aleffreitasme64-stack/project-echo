# project-echo

Jogo experimental de exploração e gerenciamento feito em JavaScript como projeto de aprendizado.

## 🚀 Funcionalidades atuais
- Sistema de exploração por escolhas
- Sistema de descanso
- Inventário funcional
- Barra de vida e energia
- Interface renderizada em tempo real
- Estrutura base para expansão do jogo

## 🛠 Tecnologias
- - JavaScript
